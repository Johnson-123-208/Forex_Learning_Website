# Forex Learning Website

This is a premium, institutional-grade Forex learning application built with React and Vite. It features a complete 16-module curriculum, interactive paper trading simulation, position sizing tools, and a professional dark-themed UI.

## Tech Stack
- React 19
- Vite
- Tailwind CSS v4 (PostCSS)
- React Router DOM
- Framer Motion
- Recharts
- Lucide React

## Setup & Running
1. `npm install`
2. `npm run dev`

This template provides a minimal setup to get React working in Vite with HMR and ESLint rules.
